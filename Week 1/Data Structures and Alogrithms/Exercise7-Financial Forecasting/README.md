Exercise 7: Financial Forecasting

Recursion is a technique where a method calls itself to solve a smaller version of the same problem.

In this exercise, recursion is used to calculate the future value of an investment based on a fixed growth rate.

Time Complexity:
O(n)

Space Complexity:
O(n)

Optimization:
The recursive solution can be optimized by using an iterative approach or by applying the direct compound interest formula. This reduces memory usage and avoids excessive recursive calls.

Conclusion:
Recursion provides a simple solution for financial forecasting, but iterative methods are generally more efficient for larger inputs.
